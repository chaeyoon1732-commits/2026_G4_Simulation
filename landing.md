# Gemini Canvas용 랜딩페이지 생성 프롬프트

아래 프롬프트를 복사하여 Gemini Canvas에 붙여넣으세요.

---

**[프롬프트 시작]**

당신은 숙련된 프론트엔드 개발자이자 UI/UX 디자이너입니다. 아래 제공된 정보를 바탕으로 현대자동차 국내사업본부 리더들을 위한 '현장 리더 AI 면담 시뮬레이터'의 랜딩페이지를 제작해 주세요.

### 1. 앱 정보
- **앱 이름**: 현장 리더 AI 면담 시뮬레이터 ✨
- **한 줄 소개**: AI 페르소나와 실전 면담으로 리더십 역량을 강화하세요.
- **핵심 기능 목록**: 
  - 다양한 성향의 AI 페르소나 시뮬레이션
  - 실시간 면담 지표(라포, 상황파악, 해결책, 몰입도) 트래킹
  - AI 기반 상세 분석 리포트 및 액션 플랜 제공
  - 개인별 면담 이력 및 역량 성장 대시보드
  - 관리자용 시나리오 및 페르소나 설정 기능
  - 로그인 없이 즉시 체험 가능한 데모 모드
- **대표 기능 (랜딩 노출용)**:
  1. **AI 페르소나 시뮬레이션**: 실제 현장 팀원의 성향을 반영한 AI와 실전 같은 대화 연습
  2. **실시간 지표 분석**: 대화 중 라포 형성 및 목표 달성 여부를 실시간으로 시각화
  3. **AI 상세 리포트**: 면담 종료 후 강점, 개선점, 추천 대화 예시를 포함한 리포트 발행
  4. **성장 대시보드**: 누적된 면담 데이터를 통해 리더십 역량 변화를 한눈에 확인
- **기타 언급 기능**: 관리자 모드, 데모 체험 모드
- **타깃 사용자**: 현대자동차 국내사업본부 현장 리더 및 관리자
- **디자인 톤**: #002C5F(현대 블루)를 메인으로 한 '신뢰감 있는', '세련된', '직관적인' 디자인

### 2. 결과물 스펙
- **기술 스택**: HTML + CSS + JS 단일 파일, Tailwind CSS CDN 사용
- **언어**: 한국어
- **반응형**: 모바일 우선(Mobile-First) 반응형 디자인
- **이미지**: 외부 이미지 URL을 사용하지 말고, 이모지(Emoji)나 Tailwind 도형/그라데이션으로 시각적 요소 대체

### 3. 섹션 구성 (순서 엄수)
1. **Hero**: 
   - 배경에 현대 블루(#002C5F) 계열의 세련된 그라데이션 적용
   - 앱 이름 + 한 줄 소개
   - "시작하기" 버튼 (새 탭에서 APP_URL 오픈)

2. **기능 소개**: 
   - 위 4가지 대표 기능을 카드 형태로 배치
   - 각 카드에는 적절한 이모지와 설명 포함
   - 섹션 하단에 "그 외에도 관리자 모드, 데모 체험 기능을 제공합니다" 문구 추가

3. **사용 방법 (3단계)**:
   - Step 1: 페르소나 및 시나리오 선택
   - Step 2: AI와 실전 면담 시뮬레이션 진행
   - Step 3: AI 분석 리포트 확인 및 역량 강화

4. **CTA 푸터**: 
   - "지금 써보기" 버튼 (새 탭에서 APP_URL 오픈)
   - 제작자 정보: 이채윤 / 국내사업인재육성팀

### 4. 코드 구현 주의사항
- 파일 최상단에 `const APP_URL = "https://2026-g4-simulation.vercel.app/";` 상수를 선언하고 모든 버튼 링크에서 이 상수를 참조할 것.
- 모든 외부 링크 버튼에 `target="_blank" rel="noopener"` 속성 부여.
- 현대자동차의 브랜드 느낌이 나도록 폰트와 색상(Blue, Gold, Gray)을 조화롭게 사용할 것.
